# Electric-cycle
The Electric Cycle Project is developed to create an eco-friendly, economical, and energy-efficient mode of transportation. Due to the increasing fuel prices and air pollution caused by conventional vehicles, electric vehicles are becoming more important in modern society. This project aims to design and develop a bicycle that operates using electrical energy along with manual pedaling. The electric cycle helps reduce fuel consumption and provides a clean transportation system for short-distance travel.

An electric cycle mainly consists of a DC motor, rechargeable battery, motor controller, throttle, chain drive, and supporting frame structure. The rechargeable battery supplies electrical power to the motor. When the rider uses the throttle or pedals the cycle, the motor rotates the wheel and helps the cycle move smoothly with less physical effort. The battery can be charged using normal household electricity, making the system simple and cost-effective.

The project works in two different modes: pedal mode and electric mode. In pedal mode, the cycle can be operated like a normal bicycle. In electric mode, the motor provides additional power for easier riding, especially on slopes and long-distance travel. Some electric cycles also use a pedal-assist system, where the motor automatically supports the rider while pedaling.

The main objective of this project is to reduce environmental pollution and promote the use of green energy technology. It also helps in reducing traffic congestion and transportation costs. Electric cycles require less maintenance compared to petrol vehicles and produce very low noise during operation.

This project is highly useful for students, office workers, delivery services, and short-distance commuters. It demonstrates the practical application of electrical and mechanical engineering concepts in modern transportation systems. The Electric Cycle Project is an innovative solution for sustainable mobility and supports the development of smart and environmentally friendly transportation methods for the future.
